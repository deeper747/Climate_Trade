# Climate_Trade

This repository contains code and data for analyzing global trade–climate interactions.
It includes scripts that:

- download World Bank data using `wbstats`
- process and join spatial data
- create static and interactive maps
- support reproducible analysis using R Markdown

## Folder Structure
Climate_Trade/
├── data/raw/ # raw downloaded datasets (never edited)
├── data/processed/ # cleaned data, ready for analysis
├── R/ # helper R functions
├── scripts/ # analysis and processing scripts
├── docs/ # rendered HTML output for GitHub Pages
├── figs/ # figures and exported images
└── Export.Rmd # main analysis notebook
